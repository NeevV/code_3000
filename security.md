# Security Overview

## Intended Users

This repository is mainly for myself, my professor, and classmates in this course. 
It contains programming assignments. 
It is not meant for commercial use or for handling real user data.

## Risk Assessment

There is no sensitive information stored in this repository. 
The code is only for class assignments. 
The main risk would be academic misuse from other students (like copying assignments), not a data breach.
There are no passwords or other security measures included.

## Security Measures

This repository does not require strong security protections because it does not contain sensitive information. If needed, the repository can be set to private and just visible to professors, however with collaborative projects this may not be possible. Also, changes can be managed through pull requests. Since this is a school project with basic code, additional security features are not really necessary.
